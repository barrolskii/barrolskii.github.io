    Project Scorch

    Project Scorch is a game I am working on in my spare time. It's a wave survival third person shooter currently in a pre-alpha stage. I've started this project to explore UE4 more in depth so I can try and broaden my skills with the engine.

    The code for this project is available on my Github so feel free to check it out if you're interested. I'll also put bullet point lists here for some of the features I want to add to the game later on.

    Broken chain is a 3D platforming collectathon made in UE4. This project consisted of myself and four other members as part of my technical games production module during my second year. This was my first time working with a team and I'm very please that we got to present this game to Ubisoft at their Leamington Spa studio.
