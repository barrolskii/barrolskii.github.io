    Store Data App

    This project is a rework from a previous university assessment from my task based software engineering module. During this assessment I had to use the TPL (task parallel library), LINQ and PLINQ to split, analyse and query over 5 million sets of data from over 10,000 individual files. In this rework instead of utilising the TPL I set out to make an improved version of the app, building a new version from the ground up, managing the threading myself and writing my own algorithms for querying data instead of relying on LINQ and PLINQ.

    The code for this project is available on my GitHub.

    Custom Launcher

    During this project I created my own program launcher similar to that of Steam and the Blizzard launcher. This program which I named the "Topaz Launcher", allows me to add programs at will, launching them from this program and adding a background image for each program that I add. My plan for this launcher is to use it as a hub for future apps and games that I make and I will add more features and functionality to it the more I use the launcher.

    The code for this project is available on my Github.
